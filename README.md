# 🐾 Atividade Extencionista - Site UniDoações

## 📌 Sobre o Projeto

O **UniDoações** é uma plataforma desenvolvida como parte de uma **atividade extencionista avaliativa**, com o objetivo de facilitar a conexão entre pessoas que desejam ajudar animais e instituições de apoio.

A plataforma busca promover a solidariedade por meio de **doações** e **solicitações de ajuda**, incentivando o cuidado e o bem-estar animal de forma acessível e organizada.

## Funcionalidades

* Página inicial informativa
* Cadastro de usuários
* Solicitação de ajuda
* Página de doações
* Página sobre o projeto
* Navegação intuitiva

##  Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
